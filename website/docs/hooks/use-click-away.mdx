# useClickAway

| 信息 | 内容 |
| --- | --- |
| 分类 | DOM 与交互 |
| 导出 | `useClickAway` |
| 源码 | [packages/core/src/use-click-away/index.ts](https://github.com/soberzen/react-use/blob/main/packages/core/src/use-click-away/index.ts) |

监听目标元素外部的点击或触摸事件。常用于下拉菜单、弹窗、Popover、右键菜单等需要点击外部关闭的组件。

## 使用

```tsx
import { useRef, useState } from 'react';
import { useClickAway } from '@sober/react-use-core';

function Dropdown() {
  const ref = useRef<HTMLDivElement>(null);
  const [open, setOpen] = useState(false);

  useClickAway(ref, () => {
    setOpen(false);
  });

  return (
    <div ref={ref}>
      <button onClick={() => setOpen(true)}>打开菜单</button>
      {open && <div>菜单内容</div>}
    </div>
  );
}
```

## 自定义监听事件

```tsx
import { useRef } from 'react';
import { useClickAway } from '@sober/react-use-core';

function ContextMenu() {
  const ref = useRef<HTMLDivElement>(null);

  useClickAway<MouseEvent>(
    ref,
    (event) => {
      console.log(event.clientX, event.clientY);
    },
    ['click', 'contextmenu']
  );

  return <div ref={ref}>菜单内容</div>;
}
```

## 参数

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `ref` | `RefObject<HTMLElement \| null>` | 必填 | 目标元素 ref。 |
| `onClickAway` | `(event: E) => void` | 必填 | 点击目标元素外部时触发。 |
| `events` | `string[]` | `['mousedown', 'touchstart']` | 需要监听的 document 事件列表。 |

## Type Declarations

```ts
function useClickAway<E extends Event = Event>(
  ref: RefObject<HTMLElement | null>,
  onClickAway: (event: E) => void,
  events?: string[]
): void;
```

## 细节

- 点击目标元素内部或其子元素时不会触发回调。
- 回调函数更新后，会执行最新的 `onClickAway`。
- 组件卸载时会自动移除 document 事件监听。
