# useDocumentTitle

| 信息 | 内容 |
| --- | --- |
| 分类 | 副作用 |
| 导出 | `useDocumentTitle` |
| 源码 | [packages/core/src/use-document-title/index.ts](https://github.com/soberzen/react-use/blob/main/packages/core/src/use-document-title/index.ts) |

设置页面的 `document.title`。当组件卸载或标题变化时，会恢复上一个标题，适合详情页、编辑页和临时流程页。

## 使用

```tsx
import { useDocumentTitle } from '@gysober/react-use-core';

function UserDetail({ name }: { name: string }) {
  useDocumentTitle(`${name} - 用户详情`);

  return <div>{name}</div>;
}
```

## 动态标题

```tsx
import { useState } from 'react';
import { useDocumentTitle } from '@gysober/react-use-core';

function Editor() {
  const [dirty, setDirty] = useState(false);

  useDocumentTitle(dirty ? '未保存的编辑' : '编辑器');

  return (
    <button onClick={() => setDirty(true)}>
      标记为未保存
    </button>
  );
}
```

## 参数

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `title` | `string` | 必填 | 要设置的页面标题。传入字符串会先执行 `trim()`。 |

## Type Declarations

```ts
function useDocumentTitle(title: string): void;
```

## 细节

- 空字符串或只包含空格的字符串不会修改标题。
- 非字符串运行时输入会被忽略。
- 组件卸载时会恢复调用前的 `document.title`。
